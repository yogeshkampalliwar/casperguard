# CasperGuard

AI Agent Security Layer built on Casper Testnet.

- Real-time Casper block height and CSPR price
- On-chain transaction fetching from Casper Testnet
- AI Agent risk assessment (approve/block agents)
- Deployed smart contract on Casper Testnet
- Animated particle UI with orbiting nodes
